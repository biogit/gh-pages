About the Pipe Dream Project

Learning by Doing: A Responsive Website on GitHub

GitHub is a popular platform for free hosting of project websites and now also of website projects. Over the years GitHub Pages added various useful features, such as traffic analytics. The purpose of this project is to learn about hosting a responsive HTML5 website on GitHub through the experience of doing it.

Specifically, the site is based on a responsive template that uses HTML5, CSS, and minimal JS. While larger frameworks can provide some additional interesting features, using frameworks current is beyond the scope of the project.

Possible "Under Construction" Problems

As a learning project, this website may include elements that are incomplete, broken, or that otherwise look/behave strangely. While the goal is to keep such elements to a minimum on the public-facing site, the occasional bugs, mistakes, and oversights are difficult to avoid when learning by doing.
